# Advanced Scenario Workshop

Splunk Observability Cloud의 고급 시나리오를 다루는 워크샵 모음입니다.

원활한 워크샵이 되도록 아래 모듈 중 원하는 시나리오를 선택하여 진행하세요.

## 포함 모듈

- **Isovalent Splunk O11y Cloud Integration** : Cilium 기반 eBPF 네트워크 관측성과 Splunk O11y Cloud를 연동하는 워크샵입니다.
- **Advanced Collector Configuration** : OpenTelemetry Collector의 고급 설정(Gateway, Filelog, 데이터 가공 등)을 다루는 워크샵입니다.
- **Real User Monitoring** : Splunk RUM을 활용해 브라우저 사용자 경험을 수집하고 분석하는 워크샵입니다.
- **Splunk Rookies Workshop for Observability** : Kubernetes 환경에서 Zero-code Instrumentation으로 인프라, APM, Log 를 수집하는 입문 워크샵입니다.
